Programação Web no Desenvolvimento de Sistemas

Programação Web 1 (PW1)

  • Introdução aos Fundamentos: PW1 introduz os alunos aos conceitos básicos de desenvolvimento web, incluindo HTML, CSS e JavaScript.
  • Desenvolvimento de Interfaces de Usuário: Os alunos aprendem a criar interfaces de usuário estáticas e responsivas usando HTML e CSS.
  • Programação do Lado do Cliente: PW1 ensina os fundamentos da programação do lado do cliente usando JavaScript, incluindo manipulação do DOM e eventos.
  • Boas Práticas de Desenvolvimento: Os alunos aprendem as melhores práticas de desenvolvimento web, incluindo organização de arquivos, estruturação de código e acessibilidade.

Programação Web 2 (PW2)

  • Desenvolvimento de Aplicações Dinâmicas: PW2 foca no desenvolvimento de aplicações web dinâmicas usando tecnologias como AJAX e frameworks de front-end como React.js ou Vue.js.
  • Comunicação com o Servidor: Os alunos aprendem a fazer solicitações assíncronas ao servidor usando AJAX e a manipular os dados recebidos.
  • Introdução a Bancos de Dados: PW2 introduz conceitos básicos de bancos de dados relacionais e como integrá-los às aplicações web.
  • Segurança e Autenticação: Os alunos aprendem sobre técnicas de segurança web, como prevenção contra ataques CSRF e XSS, e implementam sistemas de autenticação.

Programação Web 3 (PW3)

  • Desenvolvimento de Aplicações Complexas: PW3 concentra-se no desenvolvimento de aplicações web completas, desde a concepção até a implementação.
  • Frameworks de Desenvolvimento: Os alunos são introduzidos a frameworks de desenvolvimento back-end, como Node.js com Express.js ou Django, para criar aplicações robustas e escaláveis.
  • Integração de Bancos de Dados: PW3 aborda a integração de bancos de dados com a aplicação web, incluindo operações CRUD (Create, Read, Update, Delete).
  • Implantação e Hospedagem: Os alunos aprendem a implantar e hospedar suas aplicações web em servidores web, utilizando serviços de hospedagem em nuvem como AWS ou Heroku.
